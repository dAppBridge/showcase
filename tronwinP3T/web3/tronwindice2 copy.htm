Provably Fair

All Dice Rolls are Provably Fair using the following formula:

The random number is generated using a unique Server Seed (Hidden until you create a new Client Seed to prevent cheating!). This is combined with your own Client Seed and a numeric Nonce - which is incremented after each roll. An industry standard for securely generating random numbers without any risk to either party (Us or You)!

After the Rolls and after the Server Seed has been revealed (After changing your Client Seed a new Server Seed is also generated so we can reveal the previous) you can then compare the results to ensure we haven't cheated you! More information on the routines is available at: https://dicesites.com/provably-fair
